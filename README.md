# Мур-Луг 🐱

Уютная idle/merge-игра про ферму котиков для платформы Яндекс Игры.

**Играть:** откройте `index.html` в браузере — или [играйте онлайн](https://andrewvnukov.github.io/mur-lug/).

## Механики

- **Слияние** — тащите котика на такого же, чтобы получить котика уровнем выше (30 уровней пород)
- **Поглаживание** — тапайте по котикам, они дают монетки
- **Декор** — клубочек, лежанка, клумба, прудик, домик и фонарик; котики сами играют, спят и ловят рыбок
- **Улучшения** — бесконечная прокачка дохода, офлайн-заработка и шанса сияющего котика
- **Квесты дня** — три ежедневных задания и сундук за их выполнение
- **Отпустить** — если луг переполнен, перетащите котика на табличку — он уйдёт и оставит монетки

## Технически

Вся игра — один самодостаточный `index.html` (~2000 строк): Canvas2D-отрисовка, процедурные котики без спрайтов, интеграция с Yandex Games SDK (реклама, облачные сохранения) с корректным откатом на `localStorage` вне платформы.

Визуальный стиль: плоские заливки + тёмно-коричневые чернильные контуры (референс — Cats & Soup).
